About this project

This web application finds the number of patients that have been, will be or are currently being used in clinical trials pertaining to a certain disease/condition. Users can search for a condition by entering its name in the search box and clicking Submit to view the relevant results.

Results explanation

Patient type Explanation
Actual Patients currently participating in ongoing clinical trials
Anticipated Patients that will be used in future trials
Used Patients that have participated in completed trials
